Cara memasang server backup
1
Download dan ekstrak

Ekstrak ZIP pada PC backup, lalu jalankan INSTALL_SERVER.cmd sebagai Administrator.

2
Selesaikan instalasi

Installer memasang Python, Flask, SSH database, cloudflared, serta konfigurasi yang diperlukan.

3
Jalankan shortcut

Klik Jalankan Tracer Server. Tiga CMD untuk database, Flask, dan failover akan terbuka.

4
Periksa status

Server baru akan muncul sebagai Backup, Read-only, dan Standby sampai primary tidak tersedia.

Penting: gunakan shortcut yang dibuat installer. Jangan menjalankan cloudflared backup secara manual.
